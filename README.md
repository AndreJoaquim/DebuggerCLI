# DebuggerCLI
Debugger for Java applications

Java debugger using Java Reflection mechanism and javassist library.
(You need to download javassist lib in order to compile this)

## Usage:
* Info
* Throw
* Retry
* Return \<value>
* Get \<field name>
* Set \<field name> \<new value>
* Abort
